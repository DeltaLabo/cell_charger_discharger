void pid ( float  feedback,
unsigned  setpoint 
)

Function for the PI controller.

Parameters
feedbackaverage of measured values for the control variable
setpointdesire controlled output for the variable

First, this function will declare two variables, er and pi, which will be used to store the error and the proportional+integral value.

Then, it will calculate the error by substraction the feedback from the setpoint and store it in er



After, it will make sure that the error is never below ERR_MIN or above ERR_MAX


Then the proportional and integral component will be calculated.

They will be summed up and stored in pi

After, the function wil define the duty cycle, and make sure that is never below DC_MIN or above DC_MAX

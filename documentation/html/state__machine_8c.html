◆ Li_Ion_param()

void Li_Ion_param ( void  )

Function to define the parameters of the charge/discharge process fo Li-Ion chemistry.

This function will first show the general constants for Li-Ion chemistry that are:

  • Li_Ion_CV = 4200 mV
  • Li_Ion_CAP = 3250 mAh

For the charging current it will offer three options:

  • 0.2 C
  • 0.5 C
  • 1 C

After chosing charging current, the program will print it

Unless the user press ESC, in that case the program will be restarted to the STANBY state.

Now the program will show the end-of-charge current constant that is:

  • Li_Ion_EOC_curr = 100 mA

◆ Start_state_machine()

void Start_state_machine ( void  )

Function to start the State Machine.

At first, this function check which is the current cell. If cell_count is equal to '1' it will ask for user intervention to start. It will prompt the user to press 's'. If the user press it, the program will start. The user can also press 'ESC' and go to the STANDBY state. If the current cell is not '1' the program will start without the user intervention.

void main ( void  )

This is the main function of the program.

Initially the #Initialize() function is called

Then the while(1) loop enters

If the flag of the Timer0 is set then:.

  • The Timer0 period register is set to 7, which gives 252 instructions (including 2 of delay) to overflow
    That is: Instruction clock {32MHz/4} / Prescaler {128} / Counter {250} = 250Hz = 4ms of period
  • Then, the Timer0 flag is cleared
  • Then, the ADC channels are read by calling the read_ADC() function
  • Then, averages for the 250 values available each second are calculated by calling the calculate_avg() function
  • Then, the log is printed in the serial terminal by calling the log_control() function
  • If the counter in the variable count is cleared it means that 1 second has passed.
    The following tasks are excuted every second:
  1. If the chemistry is Li Ion the cc_cv_mode() function is called
  2. Then the state_machine() function is called
  • If the variable conv is set it means the converter shall be stated, then:
  1. The main relay is closed
  2. Then the control_loop() function is called
  3. If by that point the timer flag was set again and error message is printed
  • Else, the main relay is keep closed

Timing control is executed by calling the timing() function

void interrupt serial_interrupt ( void  )

This is the interruption service function. It will stop the process if an ESC or a "n" is pressed.

The function first define and initialize a variable (recep) to store the received character

If the UART reception flag is set then:

  • Check for errors and clear them
  • Empty the reception buffer and assign its contents to the variable recep
  • If recep received an ESC, then:
  1. Stop the converter by calling the STOP_CONVERTER() macro.
  2. Go to the STANDBY state.
  • Else If recep received an "n", then:.
  1. Stop the converter by calling the STOP_CONVERTER() macro.
  2. Go to the ISDONE state.
  • Else:
  1. Clear the recep variable.
